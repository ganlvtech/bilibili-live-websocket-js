# BiliBili 直播 WebSocket Node.js 示例

## 使用

修改 `bilibili-live-websocket.js` 中的 `roomid`

然后执行

```bash
npm install
node bilibili-live-websocket.js
```

如果有其他需求，可以自行修改 `ws.on('message', ...)` 的代码，处理各种推送消息。

## 说明

本程序代码由 BiliBili 直播网页端的 js 脚本分析得到。
